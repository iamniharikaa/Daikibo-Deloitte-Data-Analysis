# Daikibo Industrials — Data Analysis
## Deloitte Data Analytics Virtual Job Simulation | Forage

### Background
Daikibo Industrials is a global manufacturing company with 4 factories
across Japan, Germany, and China. This project was completed as part of
the Deloitte Data Analytics Virtual Job Simulation on Forage.

### Task 1 — Machine Downtime Analysis (Tableau)
- Analyzed one month of telemetry data from 4 factories
- Created a calculated field to measure downtime in minutes
- Built an interactive Tableau dashboard to identify the factory
  and machine types with the highest downtime

### Task 2 — Gender Pay Equality Analysis (Excel)
- Analyzed gender pay equality scores across all job roles
- Created a classification column using Excel formulas
- Classified scores as Fair, Unfair, or Highly Discriminative

### Tools Used
- Tableau
- Microsoft Excel
